# Javascript Forms
表单是一个包含表单元素的区域。

表单元素是允许用户在表单中（比如：文本域、下拉列表、单选框、复选框等等）输入信息的元素。

表单使用表单标签（&lt;form&gt;）定义。
```
<form>
...
  input 元素
...
</form>
```
## 表单标签
1. <form>	定义供用户输入的表单
2. <input>	定义输入域
3. <textarea>	定义文本域 (一个多行的输入控件)
4. <label>	定义一个控制的标签
5. <fieldset>	定义域
6. <legend>	定义域的标题
7. <select>	定义一个选择列表
8. <optgroup>	定义选项组
9. <option>	定义下拉列表中的选项
10. <button>	定义一个按钮
11. <isindex>	已废弃。由 <input> 代替。
